Student Machine Learning Engineer

Core ML Team · Remote · Part-Time (10–20 hrs/wk)

Core ML Team Remote 10–20 hrs/wk Fall Cohort

This isn't a data-labeling internship. You'll be embedded in our Core ML team designing model architectures, building the data pipelines that feed them, and running the experiments that decide what ships. You'll write production PyTorch code that gets reviewed by engineers who've trained models at scale, and you'll own real deliverables — not busywork tickets. If you want to learn machine learning by actually doing machine learning, this is that.

What You'll Actually Do

  • ✓Design and fine-tune LLM / ML architectures for real product use cases

  • ✓Benchmark model performance and run rigorous experiments & ablations
  • ✓Write clean, tested, production-grade Python & PyTorch code
  • ✓Collaborate in code reviews and pair with senior engineers on hard problems
